# Customer Churn Prediction

This project compares Logistic Regression, Decision Tree and Random Forest for predicting telecommunications customer churn.

## Dataset

IBM Telco Customer Churn dataset containing customer, contract, service and payment information.

## Project stages

- Data cleaning
- Exploratory data analysis
- Categorical encoding and feature scaling
- Model training
- Evaluation using accuracy, precision, recall, F1-score and confusion matrices
- Feature importance and coefficient interpretation

## Models

- Logistic Regression
- Decision Tree
- Random Forest

## Running the notebook

Open `churn_prediction.ipynb` in Jupyter Notebook or Visual Studio Code and run all cells from top to bottom.
